Day 3 問題（隣接一致：連続ペア検出）

小文字（a〜z）が半角スペース区切りで1行与えられます。
この中で、同じ文字が2つ以上連続して出現する箇所があればその文字を出力し、
どの文字も連続していない場合は -1 を出力してください。

入力

1行に小文字英字1文字がスペース区切りで並ぶ
例：a a b a a、x y z、m m m n n

出力

最初に見つかった「同じ文字が連続している箇所」の文字

存在しない場合は -1

例

入力：a a b a a
説明：先頭に a a が連続 → 出力：a

入力：x y z
説明：同じ文字が隣り合っていない → 出力：-1

入力：m m m n n
説明：m m が最初に出現 → 出力：m

制約

すべて小文字英字1文字

半角スペース区切り

3個以上のトークンが与えられる

ヒント

配列化して左から順に比較。

arr[i] === arr[i-1] を検出すれば終了。

計算量は O(N)。
